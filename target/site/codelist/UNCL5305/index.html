Consumption Tax Category code (Subset of UNCL5305)

Identifier
UNCL5305
Agency
UN/CEFACT
Version
D.16B
Subset
OpenPEPPOL-JP
Usage
/ ubl:Invoice / cac:AllowanceCharge / cac:TaxCategory / cbc:ID
/ ubl:Invoice / cac:TaxTotal / cac:TaxSubtotal / cac:TaxCategory / cbc:ID
/ ubl:Invoice / cac:InvoiceLine / cac:Item / cac:ClassifiedTaxCategory / cbc:ID
Codes
S
Standard rate Consumption Tax

Code specifying that the standard CT rate is levied from the invoicee.

AA
Lower rate Consumption Tax

Code specifying that the lower CT rate is levied from the invoicee.

E
Exempt from Tax

Code specifying that an item liable for Consumption Tax is exempted due to trade circumstances.

Z
Zero rated goods

Code specifying that the items have a zero rate Consumption Tax.

G
Free export item, Consumption Tax not charged

Code specifying that the item is free export and taxes are not charged.

O
Outside scope of tax

Code specifying that the seller is not registered for Consumption Tax.
